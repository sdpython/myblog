Quelques notions sur les dates

La classe StockPrices utilise les dates sous forme de chanes de caractres. De cette faon, il n'est pas possible de faire des oprations dessus. Pour ce faire, il faut les convertir en un objet appel datetime.

In [15]: 
from datetime import datetime, timedelta
dt = datetime.strptime("2014-03-31","%Y-%m-%d")
dt
Out[15]:
datetime.datetime(2014, 3, 31, 0, 0)

On ajoute un jour :

In [16]: 
delta = timedelta(1)
dt = dt + delta
dt
Out[16]:
datetime.datetime(2014, 4, 1, 0, 0)

Puis on convertit dans l'autre sens :

In [20]: 
s = dt.strftime("%Y-%m-%d")
s
Out[20]:
'2014-04-01'
