Here are a few modules I will probably use when I need them to generate documentation with Sphinx. This is just to avoid searching for them again.

  • sphinxcontrib.programoutput: easily show output of command lines.
  • sphinxcontrib.nvd3: add reactive graphs based on d3.js.
  • sphinx-gallery: compute a gallery
  • sphinxcontrib-bayesnet (based on tikz-bayesnet): display bayesian net (relies on latex and a latex extension)
  • sphinxcontrib-findanything: seems easy to try, add search functionalities
  • sphinxcontrib-googlechart: easily add graphs based on google chart, should be convenient to add maps
  • ABlog: a small framework to maintain a blog
  • Cactus: a framework to produce a static web site
And some others modules to easily build graphs with javascript and python:
  • pygal: graph and svg
  • databench: easy to way to display animated graph
